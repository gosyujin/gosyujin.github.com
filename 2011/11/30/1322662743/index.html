readlineが入っていなかったのだろうか？

$ sudo yum install readline-devel
$ cd $rvm_path/src/RUBYVERSION/ext/readline
$ ruby extconf.rb
$ make
$ make install

オプション--with-readline-dir指定は特になくていけた。
