インストール

ソースをgit clone で取得してくる。

$ git clone git://source.ffmpeg.org/ffmpeg.git ffmpeg

$ ./configure --disable-yasm
$ make
$ sudo make install

オプション

|*-y|アウトプットファイルを上書きする|

|*-n|アウトプットファイルを上書きしない|

|*-i|インプットファイル指定|

|*-acodec|オーディオコーデック(copyでコピー)|

|*-strict|厳密さ(very > strict > normal(デフォルト) > unofficial > experimental)|

うーんできることが多すぎて惑う。
