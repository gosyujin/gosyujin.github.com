環境

  • Windows XP
  • Ruby 1.9.3
    • Rubygems 1.8.11
    • Bundler 1.0.21
    • ZenTest 4.6.2
    • autotest-growl 0.2.16(だったか？)
    • rspec 2.8.0
  • Growl for Windows 2.0.9

Macに置き換えてもいけるかは後で確認。

手順

Growls for Windowsインストール

  • Growl for Windows ここから落とす。

Gem入手

  • Gemfileファイルを作って以下のgemファイルを記載
gem 'ZenTest'
gem 'autotest-growl'
  • bundle install

.autotest

  • 前回何も記載しなかった.autotestファイルに以下を追記。
require 'autotest/growl'

実行

$ bubdle exec autotest
